Stage 1 reads an article and generates a small number of GAP search queries meant to fill
genuine information gaps in the article -- each query is dispatched AS-IS to a single web
search (no splitting, no follow-up), so its wording is exactly what gets searched.

This query has ALREADY been confirmed self-contained (it does not rely on unstated context, and
does not use a vague "other X" without naming X) -- do not re-judge that. Your only job here is
to grade it using EXACTLY one of these four labels:
- good: a legitimate, fact-seeking question grounded in this article
- hallucinated_competitor: the query text ITSELF writes down a specific competitor/product/
  entity name that is NOT one of the article's own extracted entities and does not appear
  anywhere in the article text. This label requires an actual invented proper noun to be
